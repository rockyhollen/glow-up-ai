# Glow-Up for Men UI Spec

## Pages

### Landing Page `/`
Goal: Convert cold traffic into paid report purchases.

Sections:
1. Hero with product promise
2. Trust/result chips
3. Included report modules
4. How it works
5. Sample output modules
6. Pricing CTA
7. Footer

Primary CTA:
- `Get My Blueprint` -> `/api/checkout`

Secondary CTA:
- `Run local test` -> `/upload`

### Upload Page `/upload`
Goal: Collect enough data to create a personalized report.

Fields:
- Name
- Age
- Goal
- Budget preference
- Maintenance level
- Lifestyle
- Style preference
- Photos

Photo guidance:
- front-facing face photo
- side-angle face photo
- hair visible
- optional full-body photo

### Report View
Goal: Give customers a premium report they want to save, use, and share.

Sections:
1. Archetype and top transformations
2. Celebrity direction
3. Face diagnostic
4. Hair system
5. Beard system
6. Skin and self-care
7. Style system
8. Presence and execution

## Visual Identity

Tone:
- premium
- masculine
- direct
- structured

Colors:
- background: near black
- cards: deep charcoal
- accent: warm gold
- text: soft ivory

UI patterns:
- large editorial headings
- rounded cards
- diagnostic chips
- section numbers
- sticky report sidebar on desktop
